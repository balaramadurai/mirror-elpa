* Contributing

Everyone is welcome to contribute to =mirror-elpa= project. Feature requests,
proposals, issue reports, documentation improvements, code patches - just to
name few things that the project can gain from.

Please note that the main upstream of =mirror-elpa= is located on [[https://gitlab.com/d12frosted/mirror-elpa][GitLab]], and
it's advisable to send your contributions there. But they can be accepted also
via [[mailto:boris@d12frosted.io][email]] and [[https://github.com/d12frosted/mirror-elpa][GitHub]].

Happy hacking!
